A simple and aesthetic desktop timer for deep work. 

## Features
- **Floating Window**: Stays on top of your other apps so you never lose track.
- **Minimal Design**: No distractions—just you and your timer.
- **Custom Themes**: in progress

## How to Run
1. **Download**: Clone this folder to your computer.
2. **Install**: Open your terminal in the folder and type `npm install`.
3. **Start**: Type `npm start` to open the app.

## Project Files
- `index.js`: Opens the app window.
- `renderer.js`: Handles the timer logic.
- `styles/main.css`: Makes everything look beautiful.
